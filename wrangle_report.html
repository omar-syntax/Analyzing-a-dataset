Data Wrangling Report

This report provides an overview of the data wrangling efforts undertaken in the wrangle_act.ipynb notebook. The primary goal was to clean and prepare the dataset for analysis, ensuring that it was accurate, consistent, and complete.

1. Data Gathering

Data was gathered from multiple sources including a Twitter archive, an additional dataset containing tweet image predictions, and a supplementary CSV file. These were loaded into the notebook using appropriate Python libraries such as pandas.

2. Data Assessment

The gathered data was assessed both visually and programmatically to identify issues that needed cleaning. Several quality and tidiness issues were identified during this process:

  • Missing values in several columns, such as in_reply_to_status_id and expanded_urls.
  • Incorrect data types, especially in columns like tweet_id and timestamp.
  • Inconsistent or incorrect entries in categorical columns like dog stages and dog names.
  • Duplicate records and retweets that needed to be removed.

3. Data Cleaning

The identified issues were addressed through systematic data cleaning. The following steps were undertaken:

  • Missing values were handled by either filling them appropriately or removing the affected rows.
  • Data types were corrected using pandas methods to ensure proper format for analysis.
  • Inconsistent entries were standardized, and invalid entries were either corrected or removed.
  • Duplicates and retweets were filtered out to maintain only original tweets with dog ratings.
  • The dataset was restructured to ensure that all necessary columns were correctly formatted and populated.

4. Final Dataset

After the cleaning process, the dataset was verified to ensure that all issues were resolved. The cleaned dataset is now ready for further analysis and contains only accurate, relevant data. The wrangling process has significantly improved the quality and usability of the data, laying a strong foundation for subsequent analysis.

This concludes the data wrangling report. The cleaned dataset is now prepared for exploratory analysis and modeling.
